# NASA APOD App

## Description
Space by Date
- A simple web app that allows users to see the NASA APOD based off submitted date. 


## Technologies Used
* HTML5
* CSS3
* JavaScript/jQuery
* NASA APOD Database API
* Git/Github 
* Github Pages for Hosting
* Google Fonts
